# Exploratory Data Analysis and Visualization Laboratory

## Overview
This repository contains the source code, datasets, and analytical outputs for the Exploratory Data Analysis and Visualization laboratory course. The objective of these experiments is to apply statistical and visualization techniques to interpret complex datasets using the R programming language.

## Prerequisites
The experiments are implemented using R. The following software and libraries are required:

* **R Programming Language** (Version 4.0 or higher)
* **RStudio Desktop**
* **Required Libraries:** `ggplot2`, `dplyr`, `tidyr`, `lubridate`

## Usage
Each experiment directory contains:
1.  **Source Code:** An R script (`.R`) containing the data processing and visualization logic.
2.  **Dataset:** The CSV file used for analysis.
3.  **Outputs:** Generated visualization figures.
